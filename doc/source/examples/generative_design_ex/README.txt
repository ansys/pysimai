

Generative Design Tutorials
---------------------------

This section provides a collection of practical script examples illustrating the Generative Design feature of SimAI.
They serve as a reference guide for users to implement similar solutions in their projects.

Tutorial overview:

1. **Creating a Project and Uploading Training Data** - Learn how to set up a new project and upload geometry files
2. **Building a Generative Design Model** - Configure and train a model on your training data
3. **Generating Random Geometries** - Create new random design variations
4. **Interpolating Between Geometries** - Generate smooth transitions between existing designs

Follow the tutorials in order for the best learning experience.
